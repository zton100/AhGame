# 深渊遗装 World Bible v0.1

## 一句话世界观

深渊裂口污染了现实、记忆与死者遗物，玩家作为拾遗者进入污染区回收遗装、剥离魂核，并用被污染的力量对抗深渊源头。

## 玩家身份：拾遗者

玩家不是传统勇者，而是被王国、圣裁庭与各类残存势力默许存在的拾遗者。

拾遗者的职责：

- 进入深渊污染区。
- 回收被污染的遗装。
- 从怪物、亡魂与遗物中剥离魂核。
- 净化或利用深渊力量。
- 通过装备、魂核与技能构筑自己的战斗方式。

拾遗者的道德位置应保持灰色：他们阻止污染扩散，但也必须主动接触禁忌力量。

## 世界秩序

深渊裂口出现前，世界由几股力量共同维持秩序：

- 人类王国：负责城邦、军队、商路与普通民众秩序。
- 圣裁庭：负责镇压异端、审判污染、封存危险遗物。
- 古代术士：研究王国建立前的灰烬、寒霜、亡魂与星界知识。
- 亡魂王国：死者并非完全消散，古老亡国曾以仪式维持死者秩序。

深渊裂口出现后，这些秩序同时被污染。现实地貌、历史记忆、死者残响与旧时代遗物开始互相重叠，形成适合刷宝与章节推进的污染区。

## 深渊是什么

深渊不是单纯的地下世界，而是一种污染现实结构的裂口。

深渊会污染：

- 地点：城镇、墓园、教堂、王城、裂谷会变成深渊污染区。
- 记忆：旧战场、英雄传说、审判记录会以扭曲形态重现。
- 死者：亡魂、尸骸、残响会被重新驱动。
- 遗物：死去强者的装备会变成遗装。
- 信仰与术式：圣光、亡语、灰烬、寒霜等力量都会出现失控版本。

深渊适合被表现为“可刷、可深入、可腐化”的系统来源。后续深渊领域可以按污染源拆分，例如瘟疫深渊、血月深渊、灰烬深渊。

## 遗装是什么

遗装是死去强者的装备、战争遗留物或被深渊污染的禁忌遗物。

遗装的特点：

- 保留旧主的战斗记忆。
- 可能携带职业、阵营或元素倾向。
- 词缀代表遗装中残留的战技、诅咒、祝福或污染。
- 高品质遗装会更像“活着的装备”，甚至反向影响拾遗者。

遗装不是普通掉落装备的换皮，它是角色构筑的核心载体。

## 魂核是什么

魂核是生者、死者、怪物或强大遗物内部残留的战斗意志结晶。

魂核可来自：

- 被深渊污染的怪物。
- 旧时代英雄或术士的残响。
- 亡魂王国的死者仪式。
- 深渊领域的核心污染源。

魂核适合承载被动效果、构筑标签、召唤强化、技能变体或高风险收益。它与遗装的区别是：遗装偏装备与词缀，魂核偏灵魂、机制和构筑核心。

## 五个职业的世界观来源

### 流放者

被圣裁庭驱逐的暗影猎人。流放者曾被训练用于追踪污染、暗杀异端与清理不可公开的目标，但因使用毒、暗影与禁忌血术被逐出圣裁庭体系。

擅长方向：

- 毒
- 流血
- 暗影
- 暴击
- 低血爆发

命名关键词：流放、毒刃、影步、血裂、断誓、残月。

### 亡语者

能听见死者残响的人。亡语者不是单纯的死灵法师，而是亡魂王国崩塌后仍能与死者秩序沟通的异类。

擅长方向：

- 召唤
- 骸骨
- 诅咒
- 献祭
- 死者残响

命名关键词：亡语、骨仆、墓誓、残魂、骨门、死王。

### 灰烬术士

研究灰烬王城禁术的术士。灰烬术士相信火焰不是毁灭，而是将旧世界烧成可重写的灰。

擅长方向：

- 火焰
- 燃烧
- 爆炸
- 法术连锁
- 灰烬禁术

命名关键词：灰烬、余火、王冠、焚印、爆燃、烬链。

### 冰痕猎手

来自寒霜裂谷的猎手。冰痕猎手以陷阱、冰霜标记和穿透射击猎杀污染生物，他们更像荒野巡守者，而非宫廷弓手。

擅长方向：

- 冰霜
- 冻结
- 陷阱
- 穿透
- 控制

命名关键词：冰痕、霜标、裂谷、冻矢、寒牙、霜网。

### 圣裁者

旧圣裁庭遗民。圣裁者并不代表完整正义，他们继承的是审判、格挡、护盾与反击的旧秩序残片。

擅长方向：

- 圣光
- 格挡
- 护盾
- 审判
- 反击

命名关键词：圣裁、审判、盾誓、辉印、断罪、旧庭。

## 怪物阵营

### 亡骨系

- 核心视觉关键词：白骨、墓土、锈甲、空眼窝、断旗。
- 常见怪物类型：骷髅兵、亡骨守卫、墓穴弓手、骨门看守。
- 常见掉落偏向：护甲、头盔、骨系魂核、格挡或护甲词缀。
- 抗性或弱点方向：偏物理抗性，可弱圣光或火焰。
- 适合章节：墓园边境、亡骨陵寝、深渊之门前段。

### 瘟疫系

- 核心视觉关键词：腐肉、绿雾、脓包、鼠群、破布。
- 常见怪物类型：瘟疫鼠、腐化教徒、疫病尸、毒雾携带者。
- 常见掉落偏向：毒伤装备、持续伤害词缀、瘟疫魂核、匕首或手套。
- 抗性或弱点方向：偏毒抗性，可弱火焰或圣光。
- 适合章节：瘟疫教堂、腐化圣所、墓园边境后段。

### 血月系

- 核心视觉关键词：红月、血纹、剥皮旗、祭坛、黑红光晕。
- 常见怪物类型：血月信徒、血祭术士、裂肉者、月蚀猎手。
- 常见掉落偏向：流血、暴击、低血爆发、献祭类魂核。
- 抗性或弱点方向：偏流血或暗影抗性，可弱冰霜或圣光。
- 适合章节：血月荒原、瘟疫教堂后段、深渊之门。

### 灰烬系

- 核心视觉关键词：焦黑王城、余火、熔裂盔甲、灰烬风暴。
- 常见怪物类型：灰烬残魂、余火骑士、烬术士、焦骨巨兵。
- 常见掉落偏向：火焰、爆炸、法术连锁、法杖或戒指。
- 抗性或弱点方向：偏火抗性，可弱寒霜。
- 适合章节：灰烬王城、虚空边境前段。

### 寒霜系

- 核心视觉关键词：冰裂、白雾、霜刺、冻结尸体、蓝白残光。
- 常见怪物类型：霜尸、裂谷猎兽、寒霜幽魂、冻甲守卫。
- 常见掉落偏向：冰霜、冻结、陷阱、控制、弓弩或靴子。
- 抗性或弱点方向：偏冰抗性，可弱火焰或流血。
- 适合章节：寒霜裂谷、虚空边境、终末深渊前段。

### 圣裁腐化系

- 核心视觉关键词：破碎圣徽、黑化圣光、审判锁链、白金残甲。
- 常见怪物类型：腐化裁决者、断誓圣卫、黑辉审判官、锁链忏悔者。
- 常见掉落偏向：护盾、格挡、反击、圣光与负面代价并存的遗装。
- 抗性或弱点方向：偏圣光抗性，可弱暗影或深渊。
- 适合章节：圣裁废都、腐化圣所、遗装王座。

### 深渊系

- 核心视觉关键词：黑紫裂口、无形肢体、虚空眼、污染结晶、低语。
- 常见怪物类型：深渊小鬼、裂口爬行者、虚空残影、深渊使徒。
- 常见掉落偏向：深渊品质装备、强力机制词缀、负面代价、混合属性魂核。
- 抗性或弱点方向：抗性不稳定，可按领域主题变化。
- 适合章节：虚空边境、深渊之门、遗装王座、终末深渊。

## 章节推进逻辑

章节推进应服务刷宝节奏：每章有明确污染主题、怪物阵营、掉落偏向和构筑诱因。

推荐章节名：

- 墓园边境
- 瘟疫教堂
- 血月荒原
- 亡骨陵寝
- 灰烬王城
- 寒霜裂谷
- 圣裁废都
- 虚空边境
- 腐化圣所
- 深渊之门
- 遗装王座
- 终末深渊

章节设计原则：

- 前期章节应解释基础阵营与装备来源。
- 中期章节引入职业主题装备与混合阵营。
- 后期章节强调深渊污染、强力遗装和负面代价。
- 每个章节最好绑定 1-2 个主要阵营，少量混入深渊系。
- Boss 关应掉落更强烈的阵营主题遗装或魂核。

## 装备品质的世界观解释

- 普通：残破装备。仍可使用，但只保留基础形制。
- 魔法：带微弱灵性。装备中残留少量旧主习惯或污染回声。
- 稀有：拥有战斗记忆。装备会重复旧主曾经的战斗方式。
- 史诗：承载旧时代战技。装备中封存了可被拾遗者复现的战斗技巧。
- 传奇：某位强者的遗物。装备拥有明确历史、称号或旧主身份。
- 远古：古代战争遗留物。来自王国、亡魂国度或术士文明之前的大战。
- 神话：接近神明权柄。装备触及旧信仰、王权或超凡规则。
- 深渊：被深渊完全污染的禁忌遗装。力量极强，但通常伴随负面代价。

## 深渊装备为什么强大且有负面代价

深渊装备不是更高级的普通装备，而是被深渊重写过的遗装。

强力效果来源：

- 深渊放大了旧主执念。
- 装备不再遵守普通战技限制。
- 深渊污染会把不同阵营、元素和职业机制强行缝合。
- 装备可能直接借用裂口中的异界规则。

负面代价来源：

- 拾遗者会被装备反向侵蚀。
- 装备可能吞噬生命、护盾、抗性或资源。
- 强力效果可能伴随失控、诅咒、易伤、低血风险或召唤反噬。
- 深渊遗装越强，越不像工具，而像一段仍在活着的污染意志。

## 命名规则

### 装备名

推荐格式：意象 + 材质/部位/遗物。

示例：

- 血月断刃
- 瘟疫使徒之握
- 亡骨守门人的残盔
- 灰烬王冠
- 霜裂猎靴
- 黑辉审判盾

装备命名应优先表达：阵营、旧主、材质、部位、污染状态。

### 技能名

推荐格式：动作 + 元素/状态。

示例：

- 毒刃
- 血裂
- 骨仆
- 灰烬箭
- 寒霜标记
- 审判打击
- 影步割裂
- 盾誓反击

技能名应短、清晰、可在 UI 中快速阅读。

### 怪物名

推荐格式：阵营 + 类型。

示例：

- 瘟疫鼠
- 血月信徒
- 亡骨守卫
- 深渊小鬼
- 灰烬残魂
- 腐化裁决者

怪物名应让玩家一眼判断阵营和威胁类型。

### 深渊领域名

推荐格式：核心污染源 + 深渊。

示例：

- 瘟疫深渊
- 血月深渊
- 亡骨深渊
- 灰烬深渊
- 寒霜深渊
- 圣裁深渊

深渊领域名应直接绑定掉落池、怪物阵营和污染机制。

## 禁用命名风格

避免：

- 现代都市词汇，例如霓虹街区、公司、手机、网吧。
- 过于网络化的词，例如无敌、爆款、欧皇、打工人。
- 过于搞笑的名字，例如超大老鼠王、快乐小鬼。
- 纯英文堆砌，例如 Dark Abyss Blade of Power。
- 没有暗黑感的普通 RPG 名称，例如新手剑、勇者头盔、火球术大师。
- 和现实宗教或政治强绑定的敏感命名。
- 过长且难以在卡牌或战斗日志中阅读的名字。

## 配置字段建议

本版本不强制修改 JSON schema。后续可以为以下配置增加 `description` 或 `lore` 字段：

- `classes.json`
- `chapters.json`
- `monsters.json`
- `equipment_templates.json`
- `skills.json`
- `soul_cores.json`

字段用途建议：

- `description`：短描述，用于 UI 展示。
- `lore`：较长背景，用于图鉴、详情页或剧情型说明。
- `tags`：继续作为系统计算字段，避免和纯展示 lore 混用。
